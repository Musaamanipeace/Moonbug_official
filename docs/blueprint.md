# **App Name**: Moon Dial

## Core Features:

- Moon Phase Rendering: A high-fidelity visual representation of the current moon phase (waxing, waning, full, or new) for the current date.
- Celestial Clock Mapping: An interactive 24-hour clock interface that maps the moon's current position in the sky relative to the horizon.
- Offline Astronomical Engine: A standalone mathematical engine that calculates lunar cycles and positioning locally without requiring internet access or external APIs.
- Minimalist Focus Mode: A streamlined interface dedicated exclusively to moon data, removing all extraneous settings or distractions.

## Style Guidelines:

- Background: Deep Obsidian (#0A0A0B) to simulate the vastness of the night sky.
- Primary: Lunar Glow (#F5F5F5) for the moon graphic and clock indicators.
- Accent: Starlight Blue (#2C3E50) for subtle dial markings and secondary information.
- Numerical markings and time data use 'Inter' for a clean, modern, and highly legible look.
- Minimalist vector outlines for phase icons to maintain a professional, scientific aesthetic.
- A centered, fullscreen circular layout that mimics a traditional analog clock or planisphere.
- Subtle, fluid rotation animations that represent the moon's transit across the clock face.
